# Deal Evaluator Agent

Autonomous agent for evaluating micro-SaaS acquisition opportunities.

## Usage in Cursor

### Basic Evaluation

1. Open the agent folder in Cursor
2. Open Claude Code chat (Cmd+L)
3. Say: "Evaluate the deal in test/sample-input.json"

### For Real Deals

Create input file with your deal info, then ask Claude Code to evaluate it.

## Input Format

See `schemas/input.json` for structure.

## Output Format

See `schemas/output.json` for structure.

## When to Use

✅ Standard evaluations
✅ Batch processing
✅ Consistent scoring

Use interactive skill (claude.ai) for complex deals requiring discussion.
